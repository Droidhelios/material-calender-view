## Setup Project

Add this to your project build.gradle
``` gradle
allprojects {
    repositories {
        google()
        jcenter()
    }
    ext {
        appcompat = '1.3.0-rc01'
        constraintlayout = '2.0.4'
        stream = '1.2.2'
        core_ktx = '1.3.2'
    }
}
```
